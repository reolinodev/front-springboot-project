#### # 사용방법
dependencies를 설치해야 한다. npm이 설치된 상태에서 터미널에서 아래 구문을 실행 시킨다.

```
npm install
```

js나 css를 수정한 후에 웹팩에 적용을 하려면 아래 구문을 실행 시킨다.

```
npx webpack 
```
또는

```
npm run webpack
```
